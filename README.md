# ファイル一覧（Excel VBA）

フォルダの階層を無視して、指定フォルダ以下の全ファイルを**一枚の表にフラットに並べて、検索で当てる** Excel のファイル管理ツールです。

20年前、仕事のファイルが3日間見つからなかった事件をきっかけに作り、以来ずっと使い続けている実物を、そのまま公開しています。

> 解説記事（Qiita）：（公開後にリンクを記載）

## 考え方

「どこに入れたか」を思い出す必要はありません。フォルダの区切りを無視して全ファイルを平らに並べ、覚えている断片（ファイル名の一部）で検索して当てます。「フォルダ」は階層ではなく、ただの一つの列です。

## 特徴

- 走査は Windows API `FindFirstFileW`（Unicode 対応）＋ 非再帰スタック方式。手元の実測で **77,608件が約7秒**（毎秒およそ1万件）
- 一覧は「番号・ファイル名・フォルダ・更新日時・サイズ・パス」だけのシンプルな一枚の表
- ツリー表示なし、タグ付けなし、プレビューなし。削りに削った結果がこの形です

## 使い方（基本）

1. `ファイル一覧.xlsm` を、一覧を取りたい場所（例：デスクトップ）に置いて開く
2. 「抽出開始」ボタンを押す → そのフォルダ以下（サブフォルダ含む）の全ファイルが、更新日が新しい順に並ぶ
3. B列（ファイル名）・C列（フォルダ）・F列（パス）のダブルクリックで、ファイルやフォルダがそのまま開く
4. 「検索」ボタンで検索ワードによる絞り込み、「クリア」で元の一覧に戻る

詳しい説明は、ブック内の「**取説**」シートにあります。

## 主な機能

- 選択抽出（サブフォルダリストから対象を絞って抽出）
- お気に入り（共有サーバーなど別の場所を登録して抽出）
- 全検索（複数の一覧シートを横断検索）
- ファイルのコピー・転送・ファイル名変換・削除（ごみ箱）
- スタートフォルダ・コピー先・転送先・フォントは「設定」シートで変更可能

## 重要設定

「マクロの一覧」など一部の機能を使う場合は、次の設定が必要です。

```
ファイル → オプション → トラストセンター → トラストセンターの設定 → マクロの設定
→ 「VBA プロジェクト オブジェクト モデルへのアクセスを信頼する」にチェック
```

## 動作環境

- Windows ＋ Microsoft Excel（マクロ有効ブック）
- 32bit / 64bit 両対応（`VBA7` 条件コンパイル済み）
